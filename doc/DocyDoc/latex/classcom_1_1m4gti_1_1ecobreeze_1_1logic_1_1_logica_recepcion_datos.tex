\doxysubsection{Detailed Description}
Clase encargada de gestionar la lógica de recepción y procesamiento de mediciones desde el servidor. 

Esta clase se encarga de obtener datos de mediciones asociados a un usuario desde un servidor remoto, procesar las respuestas, y notificar al componente correspondiente mediante un listener.

Métodos principales\+:
\begin{DoxyEnumerate}
\item Solicitar mediciones desde el servidor para el usuario activo.
\item Procesar las mediciones obtenidas, extrayendo la más reciente.
\item Comunicar la medición procesada a través de una interfaz de callback.
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
Utiliza {\ttfamily Volley} para realizar solicitudes HTTP y {\ttfamily Shared\+Preferences} para recuperar el ID del usuario. 

Requiere un contexto válido de la aplicación y un listener para recibir los datos procesados. 
\end{DoxyNote}

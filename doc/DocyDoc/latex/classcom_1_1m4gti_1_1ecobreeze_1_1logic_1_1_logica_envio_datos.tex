\doxysubsection{Detailed Description}
Clase encargada de gestionar la lógica relacionada con el envío de datos al servidor remoto. 

Esta clase proporciona métodos para interactuar con un servidor remoto utilizando la biblioteca Volley. Permite registrar direcciones MAC, tokens de huella y notificaciones en la base de datos remota. La configuración del usuario actual se obtiene a través de {\ttfamily Shared\+Preferences}.

Métodos principales\+:
\begin{DoxyEnumerate}
\item Verificación de formato de direcciones MAC.
\item Envío de direcciones MAC detectadas al servidor.
\item Envío de tokens de huella al servidor.
\item Envío de notificaciones al servidor.
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
Se requiere acceso a un contexto Android para inicializar esta clase. 
\end{DoxyNote}

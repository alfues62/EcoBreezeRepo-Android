\doxysubsection{Detailed Description}
Clase encargada de gestionar la lógica de recepción y procesamiento de notificaciones desde el servidor. 

Esta clase se encarga de obtener las notificaciones asociadas a un usuario desde un servidor remoto, procesar las respuestas recibidas, y notificar al componente correspondiente mediante un listener.

Métodos principales\+:
\begin{DoxyEnumerate}
\item Solicitar notificaciones desde el servidor para el usuario activo.
\item Procesar las notificaciones obtenidas y almacenarlas en una lista.
\item Comunicar la lista de notificaciones procesadas a través de una interfaz de callback.
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
Utiliza {\ttfamily Volley} para realizar solicitudes HTTP y {\ttfamily Shared\+Preferences} para recuperar el ID del usuario. 

Requiere un contexto válido de la aplicación y un listener para recibir las notificaciones procesadas. 
\end{DoxyNote}
